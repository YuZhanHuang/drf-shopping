# drf-shopping
Practice DRF

## Setup Project

### local
- install dependency
    ```bash
    $ poetry install
    ```
- select poetry virtualenv in PyCharm
  `~/Library/Cache/pypoetry/vertualenvs/project_name_with_some_hash/bin/python3.x`
- run Django development server
  ```bash
  $ poetry run python3 manage.py runserver
  ```
